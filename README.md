## How to Run This Project:
- Clone it

```
$ git clone https://github.com/alfianadityads/ALTA-QE12-Team1-Todoist.git
```

- Go to directory

```
$ cd ALTA-QE12-Team1-Todoist
```

- Run the project

```
$ mvn clean verify
```
